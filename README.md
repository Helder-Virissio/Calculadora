## 📖 Sobre o projeto

Esta calculadora foi desenvolvida com o objetivo de praticar conceitos de desenvolvimento web, utilizando **HTML, CSS e JavaScript**.

O principal foco do projeto é a utilização de **módulos JavaScript**, aplicando os conceitos de **export** e **import** de funções entre diferentes arquivos.

---

## 🚀 Funcionalidades

- ➕ Soma  
- ➖ Subtração  
- ✖️ Multiplicação  
- ➗ Divisão  
- 🧹 Limpar visor  
- ⚡ Estrutura modular com JavaScript
